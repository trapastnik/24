# Звук МТК-24 — сэмплы

`audio.js` ищет эти файлы и играет их (петли + one-shot). Если файла нет — мягкий
откат на синтез. **Положи сюда royalty-free аудио с ТОЧНО такими именами — заиграет сразу
(перезагрузить страницу).**

| Файл | Тип | Что | Длина |
|---|---|---|---|
| `wind.mp3` | петля (loop) | ветер / холодная ночная атмосфера Петрограда | 10–30 с, бесшовная |
| `crowd.mp3` | петля | дальний гул толпы (тише; громче в массовых кадрах/штурме) | 10–30 с, бесшовная |
| `drone.mp3` | петля | низкая тревожная подложка / дроун (опц., можно не класть) | 20–60 с |
| `boom.mp3` | one-shot | холостой выстрел носового орудия «Авроры» (пушка/залп) | 1–3 с |
| `telegraph.mp3` | петля | радио/телеграф — звучит ВСЮ трансляцию на «Телеграфе», без наложения | 1–4 с, бесшовная |
| `footsteps.mp3` | петля | шаги Ленина на маршруте к Смольному (мерная ходьба) | 2–6 с, бесшовная |

**Формат:** mp3 (или m4a/wav). НЕ ogg — Safari его не всегда играет. Моно — ок.
Петли (`wind/crowd/drone`) должны быть **бесшовными** (без щелчка на стыке).

## Где взять бесплатно (CC0 / без атрибуции)
- **Pixabay** — https://pixabay.com/sound-effects/ (лицензия Pixabay, без атрибуции).
  Запросы: `wind ambience`, `crowd murmur` / `crowd ambience`, `cannon shot` / `cannon fire`,
  `telegraph key` / `morse click`, `dark drone` / `tension drone`.
- **Mixkit** — https://mixkit.co/free-sound-effects/ (free).
- **Freesound** — https://freesound.org/ (фильтр лицензии **CC0**).
