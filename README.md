# МТК №24 · «Ленин в октябре»

Динамическая объёмная карта Петрограда: маппинг-проекция действий большевиков
в октябре 1917 года для захвата власти — в хронологии, от подполья к Совнаркому.

Прототип в стиле остальных МТК музея В.И. Ленина: **vanilla JS + canvas 2D**,
без бандлеров и фреймворков, брендовая палитра (paper / brass / signal-red /
graphite) и шрифты (Nolde, 20 Kopeek).

## Что это

Наклонная **объёмная карта** исторического центра Петрограда (плита города,
вырезанная Невой и каналами, с экструдированными зданиями-ориентирами).
По таймлайну ≈ **3:21** (≥ 180 с по ТЗ) разворачивается 13 глав восстания:
камера ведётся по ключевым точкам, подсвечиваются ориентиры, анимируются
переброски сил, выстрел «Авроры», штурм Зимнего. Снизу — дикторский титр
(дата по старому/новому стилю, заголовок, подводка), справа — цитаты В.И. Ленина
с источниками (ПСС).

Формат — горизонтальная панорама 16:9 (рассчитан на проекцию > 3840×2160).

## Запуск

Данные загружаются `<script>`-обёрткой (`data/petrograd_osm.js`), поэтому работает
и просто открытием `index.html`, и через сервер:

```bash
cd BMK/24
python3 -m http.server 8124      # → http://localhost:8124
```

Управление: **пробел** — пауза/пуск, **←/→** — по главам, клик по таймлайну — перемотка.

## Откуда геоданные

Готовой векторной карты восстания 1917 в открытом доступе нет (только растровые
сканы). Поэтому база — **OpenStreetMap (ODbL)**: исторический центр Петербурга
(Нева, острова, каналы, Невский, набережные) с 1917 г. практически не изменился,
что даёт точную и редактируемую векторную основу.

`tools/fetch_osm.py` тянет через Overpass API три слоя в `data/petrograd_osm.json`:

| слой        | что                                                                 |
|-------------|---------------------------------------------------------------------|
| `water`     | `natural=water` + русла — Нева и каналы (главный герой карты)        |
| `roads`     | магистрали, набережные, мосты через Неву                            |
| `landmarks` | контуры зданий-ориентиров 1917 г. (по координатам адресов, point-in-polygon) |

Перегенерация:

```bash
python3 tools/fetch_osm.py        # → data/petrograd_osm.json
# затем обёртка для file://:
python3 -c "import json;d=json.load(open('data/petrograd_osm.json'));open('data/petrograd_osm.js','w').write('window.MTK24_OSM='+json.dumps(d,ensure_ascii=False,separators=(',',':'))+';\n')"
```

Ориентиры: Смольный, Зимний дворец, Петропавловская крепость, Мариинский и
Таврический дворцы, Главпочтамт, телефонная станция, Госбанк, вокзалы
(Николаевский, Балтийский, Финляндский). Точечные маркеры (без контура):
квартира М.В. Фофановой, позиция «Авроры» 1917 г., электростанция.

## Структура

```
index.html              разметка и UI-хром
styles.css              палитра, шрифты, лоуэр-терд, таймлайн
map24.js                рендер объёмной карты + движок таймлайна/эффектов
data/timeline.js        хронология октября 1917 (тексты и цитаты из ТЗ)
data/petrograd_osm.js   геоданные OSM (обёртка-глобал для file://)
data/petrograd_osm.json исходный JSON геоданных
tools/fetch_osm.py      загрузчик геометрии из Overpass API
fonts/                  брендовые шрифты (Nolde, 20 Kopeek, 21 Cent)
```
